# StanVoyage - campus-only platform to find travel buddies.

A web-based platform developed exclusively for **Stanley College students** to help them find compatible travel companions based on route, date, and time preferences. The application focuses on secure access, ease of use, and efficient trip coordination within the campus community.

---

## Project Overview

Travel Buddy Finder is designed to simplify travel planning among students by allowing them to:
- Create and manage travel plans
- Find other students traveling on similar routes and timings
- Coordinate trips in a safe, verified, and user-friendly environment

The system ensures that only authorized Stanley students can access the platform, promoting trust and safety among users.

---

## Key Features

- **Exclusive Access**  
  Only Stanley students can register and use the application.

- **Trip Management**  
  Users can create, view, and delete their travel plans easily.

- **Flexible Locations**  
  Users can enter their own source and destination locations instead of selecting from predefined options.

- **Time Slot Based Planning**  
  Travel timings are selected using readable time slots (e.g., *Morning (6am–12pm)*), making coordination easier.

- **Similar Trip Matching**  
  Displays trips with matching routes, dates, and timings to help users find suitable travel partners.

- **Secure Authentication**  
  Includes login, session handling, and logout functionality for secure usage.

---

## Tech Stack (High Level)

- **Frontend:** Next.js, React
- **Backend:** Node.js (API Routes)
- **Database:** MongoDB
- **Authentication:** JWT-based authentication
- **Deployment Ready:** Designed to be easily deployed on platforms like Vercel
